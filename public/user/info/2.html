实时API调用说明
请求地址：http://api.szvone.cn/api
请求方式：POST/GET
参数说明:
传输参数 参数类型 参数说明
key 字符串 授权Key，在API列表获取
city 字符串 使用URL编码后的城市名 如 %E5%8C%97%E4%BA%AC
返回数据说明:
返回参数 参数类型 参数说明
state 整数 api调用结果（200：调用api成功，400：调用api失败））
msg 字符串 api调用结果说明
api_res 数组 api调用结果
返回参数 参数类型 参数说明
code 整数 api执行结果（1：执行成功，-1：执行失败，检查API传参，0：城市名错误或不支持此城市））
now 数组 实时天气信息
返回参数 参数类型 参数说明
date 字符串 当天日期
fengli 字符串 风力
fengxiang 字符串 风向
type 字符串 天气
wendu 字符串 实时温度
msg 字符串 温馨提示
5day 数组 五天天气预报
返回参数 参数类型 参数说明
date 字符串 当天日期
fengli 字符串 风力
fengxiang 字符串 风向
type 字符串 天气
high 字符串 最高温度
low 字符串 最低温度
