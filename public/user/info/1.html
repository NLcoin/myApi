图床API调用说明
请求地址：http://api.szvone.cn/api
请求方式：POST/表单提交
参数说明（POST）:
传输参数 参数类型 参数说明
key 字符串 授权Key，在API列表获取
img 字符串 Base64编码后的图片字符串
参数说明（表单）:
传输参数 参数类型 参数说明
key 字符串 授权Key，在API列表获取
file 文件流 图片文件流
返回数据说明:
返回参数 参数类型 参数说明
state 整数 api调用结果（200：调用api成功，400：调用api失败））
msg 字符串 api调用结果说明
api_res 数组 api调用结果
返回参数 参数类型 参数说明
code 整数 api执行结果（1：执行成功，-1：执行失败，检查API传参））
img_url 字符串 图片地址
